# Machine Learning-Driven Inverse Characterization of Polyolefins from Vibrational Spectroscopy

This repository contains scientific machine learning workflows for predicting polymer physical properties directly from near-infrared (NIR) spectroscopy measurements. The project focuses on inverse characterization of heterogeneous polymer systems using interpretable and uncertainty-aware machine learning methods.

The framework integrates spectroscopy, multivariate analysis, and physically informed ML to predict key polymer properties including density, crystallinity, and short-chain branching directly from high-dimensional spectral data. The work also explores interpretable spectral representations and chemically meaningful feature attribution for understanding structure–property relationships in complex materials systems.

Applications include adaptive materials characterization, spectroscopy-driven scientific inference, and property-based plastic sorting for circular-economy workflows.
